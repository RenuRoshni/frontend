# Tourism Website

This is a simple static tourism website showcasing popular tourist destinations in India. 
It provides users with information about various places of interest, including detailed views for each destination.

## Features

- **Responsive Design:** Utilizes Bootstrap for a responsive and mobile-friendly layout.
- **Carousel:** Provides a carousel feature for detailed views of tourist spots.
- **Navigation:** Enables seamless navigation between different sections of the website.
